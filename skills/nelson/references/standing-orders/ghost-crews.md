# Standing Order: Ghost Crews

Do not allow multiple agents to unknowingly work on the same deliverable.

**Symptoms:**
- Parallel outputs from different ships overlap significantly in content or function.
- Merge conflicts appear on files not explicitly assigned to multiple captains.
- Redundant validation or duplicate solutions emerge during quarterdeck review.

**Remedy:** The admiral must maintain clear file ownership in the battle plan so that no deliverable is claimed by more than one ship. When duplicate work is discovered, halt the later effort immediately, retain the more advanced output, and reassign the freed captain to unblocked work. Quarterdeck checkpoints should include a brief ownership review to catch ghost crews early.
